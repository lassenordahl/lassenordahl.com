On a trip to New Hampshire to hike Mt. Washington my friends and I were playing a game where we ranked every object and concept against each other.

It was a bit hard to remember our numbers in a row, so I wanted to try making a jackbox-esque party pack for it. I'll prob branch the backend off into something else so I can serve multiple of these as I think the game mechanic of using a QR code is pretty fun.
